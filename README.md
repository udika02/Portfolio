# Personal Portfolio Website

A fully responsive personal portfolio website built with **HTML**, **CSS**, and **JavaScript**.  
It highlights my skills and projects as a **Web Developer**, **Data Analyst**, **Graphic Designer**, and **AI/ML Enthusiast**, with sections for About, Resume, Projects, and Certificates — all in a modern, interactive UI without using any frameworks.

## 🔧 Tech Stack
- HTML5  
- CSS3 (custom themes & variables)  
- JavaScript (DOM manipulation, interactivity)

## 📂 Features
- Responsive Design  
- Sidebar Navigation  
- Categorized Portfolio  
- Animated UI  
- No frameworks used
